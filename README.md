# The Red Garden

A modern, responsive website for The Red Garden, Patna's premier destination for weddings, receptions, and celebrations.

## About

The Red Garden is Patna's premier venue for hosting memorable events including wedding ceremonies, reception parties, birthday celebrations, engagement functions, corporate events, and anniversary parties.

## Features

- **Responsive Design**: Optimized for all devices
- **Event Gallery**: Showcase of past events and facilities
- **Booking System**: Easy online booking and inquiries
- **Admin Panel**: Management interface for venue operations
- **Contact Integration**: Direct contact forms and WhatsApp integration

## Technologies Used

- **Frontend**: React with TypeScript
- **Styling**: Tailwind CSS with shadcn/ui components
- **Build Tool**: Vite
- **Backend**: Supabase for database and authentication
- **Deployment**: Ready for modern hosting platforms

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd theredgarden-main
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:8080](http://localhost:8080) in your browser.

## Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── ui/             # shadcn/ui components
│   ├── layout/         # Layout components (Header, Footer, etc.)
│   ├── home/           # Home page sections
│   └── admin/          # Admin panel components

├── hooks/              # Custom React hooks
├── lib/                # Utility functions
```
